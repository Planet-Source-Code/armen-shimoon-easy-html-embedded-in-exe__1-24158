HTML Embedded in Executable

 

This shows you how to create a program with an internal .html file.

 

INSTRUCTIONS

1) Make a webpage. Open it up in notepad.

2) Copy the code to the clipboard.

3) In a resource editor, make a string table. Add the copied code into a string.

4) Save as .res. Load this file up in your VB project.

5) Use my function to load the webpage into a temporary file.
